Процесс установки:

1. Скопировать содержимое папки app в DocumentRoot на сервере.
2. В конфигурации сервера добавить директиву AllowOverride All
3. В папке app/config/database.php заполнить необходимые поля:
	$DB_USER = Вставить пользователя с правами доступа для создания баз данных на сервере,
	$DB_PASSWORD = Пароль
	$DB_INSTALL = Путь к серверу MySQL
	$DB_NAME = название базы данных
4. Зайти по адресу сайта, приложение автоматически создаст и настроит базу данных.

Если запускать с помощью PHP local server, то нужно зайти в директорию app и прописать следующую команду:

php -S http://localhost:8888 router.php

Сервер MySQL должен быть доступен по адресу прописанному в $DB_INSTALL
Приложене выделяет для себя отдельную базу данных и работает с ней.
